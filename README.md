# dotfiles

> Default dotfiles for Percy.

```sh
rotz clone https://github.com/kecrily/dotfiles.git
rotz install
rotz link
```

## Summary

- Shell: [zsh](shell/zsh)
- Encryption: [GnuPG](tool/gpg)
- Version control: [Git](tool/git)
- Code editor: [VS Code](editor/code)
